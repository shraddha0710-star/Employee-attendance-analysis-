# 📊 Employee Attendance Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing employee attendance data to identify patterns, trends, and workforce insights.
The dashboard provides clear visualization of attendance performance, overtime distribution, and departmental analysis to support better decision-making.

## 🎯 Objectives
- Analyze employee attendance and absenteeism
- Identify top-performing employees
- Evaluate overtime workload across departments
- Monitor late arrivals and shift-wise attendance
- Provide actionable insights for workforce improvement

## 📁 Dataset Description
The dataset includes the following fields:
- Employee ID
- Department
- Shift (Morning, Evening, General)
- Attendance Status (Present/Absent)
- Late Days
- Overtime Hours

## 📊 Dashboard Features

### 🔹 KPI Cards
- Total Employees: 200
- Total Overtime Hours: 2056
- Average Attendance: 81.5%
- Total Absent Days: 814

### 🔹 Visualizations
- Top 10 Employees by Attendance
- Department vs Overtime Hours
- Late Days by Department
- Shift vs Attendance
- Present vs Absent Ratio
- Department vs Attendance

## 📈 Key Insights
- Overall attendance rate is 81%, indicating good workforce presence
- Some departments (like HR & Finance) show higher overtime, suggesting workload imbalance
- Morning shift has the highest attendance rate
- Late arrivals are more frequent in certain departments
- Top employees maintain consistently high attendance

## 💡 Recommendations
- Introduce employee reward programs for high attendance
- Optimize workload distribution across departments
- Improve punctuality policies
- Implement shift optimization strategies
- Conduct employee feedback analysis to reduce absenteeism
